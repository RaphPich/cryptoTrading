# Configuration

## Coinbase

  - Create a Coinbase account first at <a href="https://www.coinbase.com/fr/" >Coinbase </a> 
  - Enable API key and keep the keys
  - Wait for 48 hours for activate of your API key.

## Please add credential file as follow
  - CryptoTrading/pass.json
  - {"API_SECRET": <API_SECRET>, "API_PASS": <API_PASS>, "API_KEY": <API_KEY>}

<a href="https://docs.pro.coinbase.com">Reference Coinbase API </a>
